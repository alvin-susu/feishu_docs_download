## 🎉 重大Bug修复

### 问题：账号加载失败
**症状**:
```
✅ 账号数据加载成功
Error: 账号不存在 at BotAccountManager.getAccount
```

**根本原因**:
- `saveAccounts()` 方法使用 `Array.from(this.accounts.entries())` 保存数据
- 这会创建 `[key, value]` 对的数组格式
- 但原始 accounts.json 文件使用的是普通对象数组格式
- `loadAccounts()` 中的 `new Map(loadedData.accounts)` 无法正确加载普通对象数组

**修复方案**:
修正 accounts.json 数据结构为 `[accountId, accountObject]` 对的格式：

```json
{
  "accounts": [
    [
      "bot_1772977808268_test",
      {
        "id": "bot_1772977808268_test",
        "name": "测试机器人",
        ...
      }
    ]
  ],
  "currentAccountId": "bot_1772977808268_test",
  "version": 1
}
```

## ⚠️ 权限配置提示

当前机器人可能缺少以下飞书开放平台权限：

### 必需权限
```
wiki:wiki:readonly        - 读取知识库
wiki:node:write          - 写入知识库节点
docx:document:readonly   - 读取文档
docx:document:create     - 创建文档
im:message:send_as_bot   - 发送消息
```

### 配置方法
1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 进入应用管理 → 权限管理
3. 搜索并添加上述权限
4. 保存并发布
