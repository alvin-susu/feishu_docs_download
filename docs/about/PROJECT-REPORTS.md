## 📊 项目概述

### 基本信息
- **项目名称**: 飞书知识库智能助手
- **版本**: v2.0.0
- **类型**: OpenClaw Skill 插件
- **技术栈**: Node.js 16+, Express, 飞书开放平台API
- **开源协议**: MIT License

### 项目定位
企业级飞书知识库智能化管理工具，通过OpenClaw与飞书机器人双向联动，实现：
- 🎯 智能文档定位
- 🔍 高效内容检索
- 📁 自动智能归档
- 🤝 会议协作自动化
